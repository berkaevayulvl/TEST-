# Калькулятор семейного бюджета

Это простой статический сайт:

- `index.html` — небольшой лендинг с описанием и кнопкой запуска.
- `калькулятор.html` — приложение бюджета (доходы/расходы, CSV‑экспорт, графики, прогноз, темы, валюта).

## Как запустить

Самый простой способ — открыть `index.html` двойным кликом в браузере.

Если вы используете локальный сервер (например, Live Server в VS Code/Cursor), откройте `index.html` через него — так удобнее обновлять страницу во время разработки.

## Данные

Данные операций и настройки (тема/валюта) сохраняются в `localStorage` браузера на этом компьютере.
